QUOIN — ACKNOWLEDGEMENTS

Quoin is a native macOS app, built almost entirely on Apple's system
frameworks (SwiftUI, TextKit 2, CoreText, CoreGraphics). A few open-source
Swift packages ship inside it, acknowledged below.

Quoin, MermaidKit, and Vinculum are © 2026 2389 Research, Chicago, Illinois.

    Apache License 2.0, with the Runtime Library Exception.
    https://github.com/swiftlang/swift-markdown

Sparkle
    The auto-update framework — Quoin's only network activity.
    © the Sparkle Project contributors. MIT License.
    https://github.com/sparkle-project/Sparkle

MermaidKit
    Native Mermaid diagram parsing, layout, and rendering — no browser,
    no JavaScript. First-party, developed alongside Quoin.
    © 2026 2389 Research.
    https://github.com/2389-research/MermaidKit

Vinculum
    Native LaTeX math parsing and typesetting via CoreText — no web view.
    First-party, developed alongside Quoin.

    https://github.com/2389-research/Vinculum
