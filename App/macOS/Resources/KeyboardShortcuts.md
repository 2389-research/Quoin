# Keyboard Shortcuts

Every shortcut Quoin defines, grouped by task. This is a live document —
edit it, or keep it open in a tab as a cheat sheet. Nothing here overrides a
system shortcut (⌘P still prints, ⌘H still hides).

## Files & windows

| Action | Shortcut |
| :--- | :--- |
| New document | ⌘N |
| New window | ⇧⌘N |
| Open… | ⌘O |
| Close tab | ⌘W |
| Close window | ⇧⌘W |
| Settings… | ⌘, |

## Editing a block

Quoin renders your document; you click a block to reveal and edit its source.

| Action | Shortcut |
| :--- | :--- |
| Edit source / Done editing | ⌘↩ |
| Bold | ⌘B |
| Italic | ⌘I |
| Strikethrough | ⇧⌘X |
| Inline code | ⌃⌘E |
| Highlight | ⇧⌘H |
| Add link | ⌘K |
| Toggle checkbox | ⌃⌘↩ |
| Move block up / down | ⌥⌘↑ / ⌥⌘↓ |
| Heading level 1–6 | ⌥⌘1 … ⌥⌘6 |

## Find & replace

| Action | Shortcut |
| :--- | :--- |
| Find in document… | ⌘F |
| Find & replace… | ⌥⌘F |
| Find next / previous | ⌘G / ⇧⌘G |
| Search library… | ⇧⌘F |

## Review

| Action | Shortcut |
| :--- | :--- |
| Add comment… | ⇧⌘M |
| Suggest replacement… | ⇧⌘R |
| Suggest edits (mode) | ⌃⌘R |

## Navigation

| Action | Shortcut |
| :--- | :--- |
| Quick open (empty = recents) | ⇧⌘O |
| Today's note | ⌘D |
| Back / forward through jumps | ⌘[ / ⌘] |
| Show next / previous tab | ⌃⇥ / ⌃⇧⇥ |
| Select tab 1–9 | ⌘1 … ⌘9 |

## View

| Action | Shortcut |
| :--- | :--- |
| Show/hide sidebar | ⌘0 |
| Show/hide outline | ⌥⌘0 |
| Typewriter scrolling | ⌥⌘T |
| Zoom in / out | ⌘= / ⌘- |
| Actual size | ⌃⌘0 |

*Focus Mode has no default chord — reach it from the **View** menu or the
toolbar. Assign a key from **System Settings ▸ Keyboard ▸ Shortcuts** if you
use it often.*
